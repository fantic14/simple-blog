# Simple Blog - Frontend

This is the React-based frontend for the Simple Blog application. It is built with **Vite**, **TypeScript**, and **Tailwind CSS 4**.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/) (Recommended)

### Installation
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the development server:
   ```bash
   pnpm dev
   ```

## Tech Stack
- **React 19**: Modern UI development.
- **Tailwind CSS 4**: Next-gen styling with zero-runtime CSS.
- **React Router 7**: Sophisticated client-side routing.
- **Axios**: Promised-based HTTP client for API interaction.

## Structure
- `src/pages`: Main application views (Home, Login, Register).
- `src/components`: (Coming soon) Reusable UI components.
- `src/api`: Axios instance and API configuration.
